Filter Examples

Number filter -- Display the number in a comma seperated manner and round it upto 3 decimal places

Num1 : {{num1}}

Num2 : {{num2}}

Num1 : {{num1 | number}}

Num2 : {{num2 | number}}

Num3 : {{num3 | number}}

Num2 with only 1 digit after the decimal point : {{num2 | number:1}}

Num3 : {{num3 | number:3}}

Num2 as an integer : {{num2 | number:0}}

Num3 as an integer : {{num3 | number:0}}

5.3 as an integer : {{5.3 | number:0}}

Currency filter

Takes in 2 optional arguments:1. symbol or an identifier; 2: number of digits after the decimal; default : $,number of digits : 2;

Num1 : {{num1| currency}}

Num1 in rupees : {{num1| currency:'Rs.'}}

Num1 in rupees : {{num1| currency:'USD$'}}

Num2 in rupees : {{num2| currency:'Rs.':4}}

Num2 in rupees - default : {{num2| currency}}

Name in uppercase :{{"asha" | uppercase}}

Name in lowercase :{{"ASHA" | lowercase}}

Name in uppercase :{{name | uppercase}}

Name in uppercase :{{"asha meHta" | uppercase}}

Number in uppercase:{{1234 | uppercase}}

String with currency filter : {{name | currency}}

1000ms since the epoch : {{1000 | date}}
